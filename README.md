# cinemabarredura
 Este proyecto pertenece a Amaranta Chikiframe.

 Consiste primordialmente en un sistema que toma variables ambientales y las traduce en dibujos sobre pelicula de cine.

## Hardware

Se requiere el siguiente Hardware

- Raspberry pi
- Memoria micro SD 32GB
- Power Bank
- Ramps
- Sistema de dibujo

## Software

Se requiere lo siguiente

- Raspberry Pi Imager
- Raspberry Pi OS Trixie
- Maria DB
- Grafana
