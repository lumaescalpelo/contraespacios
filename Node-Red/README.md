# Node-Red

En este documento se describen las acciones a realizar en Node-Red

- Durante el proceso de instalación, se estableció el nombre de usuario como `cinema` y la contraseña como `barredura`. La frase de encriptación se estableció como `cinemabarredura`.
- Importa el archivo `flows.json` para agregar el flow con las funciones necesaras
- Posiblemente tengas que agregar las contraseñas de los nodos mysql

## Notas

Para que esto funcione, debes tener instalado Node-Red y debes tener agregados los nodos externos.

### Nodos externos
Para instalar los nodos externos, dirigete a Manage Palette y en la sección install, busca e instala los siguientes nodos.

- @flowfuse/node-red-dashboard.
- node-red-node-mysql
